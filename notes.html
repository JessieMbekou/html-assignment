Overview

Here is a summary of some HTML tags covered during the Web Development class.

The "p" tag
This is the paragraph tag used to insert paragraphs or group of text or content in a webpage.
The "h1" tag
This is one of the headings tags. It is the biggest heading tag and generally used as title in a web page
The "header" tag
The header tag is the top section of a webpage. It usually contains introductory content or title and navigation links
The "img" tag
This is the image tag and is used to embed images in a webpage. Must contain "src" and "alt" attributes for proper functioning.
The "a" tag
This tag is used to insert links to other pages, email addresses, files or sections in a webpage. Must contain an "href" attribute to function properly.
